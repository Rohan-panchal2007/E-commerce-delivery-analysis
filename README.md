#  E-Commerce Delivery Analysis

An end-to-end data analysis project exploring delivery performance, customer satisfaction, and order trends across multiple e-commerce platforms using **Python, MySQL, SQL, and Power BI**.

---

##  Project Overview

This project performs a complete data analysis workflow on an e-commerce delivery dataset. It focuses on identifying key factors affecting delivery efficiency, customer satisfaction, and refund behavior.

The workflow includes:

* Data cleaning using Python
* Structured storage in MySQL
* Business analysis using SQL
* Interactive dashboard creation in Power BI

---

##  Objectives

* Analyze delivery performance across platforms
* Identify the impact of delivery delays on customer ratings
* Evaluate product category trends
* Study relationship between order value and satisfaction
* Build an interactive dashboard for business insights

---

---

##  Dashboard

The Power BI dashboard provides:

* KPI cards (Total Orders, Avg Delivery Time, Rating, Refund %)
* Platform comparison visuals
* Delivery delay impact analysis
* Customer feedback insights
* Order category distribution

---

##  Key Insights

*  Delivery delays significantly reduce customer satisfaction
*  Higher delays are associated with increased refund requests
*  Medium-value orders dominate overall order volume
*  Faster delivery does not always guarantee better ratings
*  Order value has limited impact on customer satisfaction
